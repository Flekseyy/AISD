Реализовать кастомный линейным список (если знаем generics, то используем, иначе для типа int):
1. Реализовать класс узла линейного списка
2. Реализовать сам класс линейного списка, который будет включать 
	- возможность создания пустого списка
	- возможность создания списка с входным параметром в виде целого числа (список из 1 элемента должен получиться)
	- возможность создания списка с входным параметром в виде массива целых чисел
	- добавление значения (по умолчанию в конец)
	- добавление значения в начало
	- добавление элемента на какую-то позицию (позиции от 1)
	- добавления диапазона значений (AddRange)
	- вывод линейного списка на экран (хочу красиво со стрелочками)
	- удаление элемента с конца
	- удаление элемента на какой-то позиции
	- удаление начиная с какой-то позиции
	- удаления все вхождения какого-то значения
	- удаление второго элемента
	- перевернуть линейный список за 1 проход
	- размер линейного списка
	- проверка на пустоту
	- проверка на наличие какого-то элемента
	- выдать индексы вхождения некоторого значения 

На каждый метод (это касается вообще всего) написать тесты или хотя бы вызовы. 
Тесты должны включать положительный результат, отрицательный, все граничные значения 

П.С. На контрольной могу дать любой вариант похожего задания, например, найти сумму всех значений элементов и тп
Линейный список может содержать указатель только на головной элемент, а может на головной и хвостовой. 
На контрольной может быть любой вариант

3. Аналогичным образом уметь реализовывать двунаправленный линейный список

4. Уметь реализовывать стек (на основе list, linkedlist, array)
	- Push: кладём новый элемент поверх всех
	- Pop: забираем верхний элемент и удаляем его из стека
	- Peek: просто смотрим, что сверху, не удаляя

5. Уметь реализовывать очередь (на основе list, linkedlist, array). Уметь реализовывать в обе стороны.

6. Уметь реализовывать дек (на основе list, linkedlist, array).
